# Task Summarizer

สรุปและวิเคราะห์รายการงาน (Task Summary & Analysis)

## Features
- สรุปงานตามกลุ่ม (status, priority, assignee, date)
- วิเคราะห์รูปแบบงาน (pattern analysis)
- สร้างรายงานประจำวัน (daily report)
- คำนวณสถิติ (metrics)

## Functions
- `summarize_tasks()` - สรุปรายการงาน
- `analyze_task_patterns()` - วิเคราะห์รูปแบบ
- `generate_daily_report()` - รายงานประจำวัน
